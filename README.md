# HAM GLOBAL WORDS

Site web moderne pour une agence de services linguistiques et solutions digitales.

## Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- React Hook Form + Zod
- Resend pour les emails

## Installation
```bash
npm install
cp .env.local.example .env.local
npm run dev
